mini_server & client 使用文档

服务器端：
运行server文件开启服务器，进入服务器功能选单：1.发送公告  2.关闭服务器

客户端：
在GEC6818开发板上运行client程序，连接上服务器后，服务器自动发送当前ID号给client端，

与好友通讯时只需要输入“【对方ID号】：聊天内容”即可。

通过触摸屏点击访问：当前好友列表，广州天气，每日笑话等功能。
